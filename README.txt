## 1. 사용층

- 설정한 사용자는 누구이며, 해결하려는 구체적인 문제점/불편함은 무엇인가요?

컴퓨터 기술 활용이 어려운 수강자 혹은 강의 내용을 쉽게 따라가지 못하는 수강자

실제 강의 환경에서는 컴퓨터 기술 활용이나 설정에 익숙지 않은 사람이 있을 뿐더러, 강의 내용의 이해 속도가 느려 실강에서 이해하지 못한 부분을 듣느라 따라가기 벅찬 사람들도 있는 등 다양한 페인이 존재했습니다. 
실제로 수업 도중 강사는 강의 일정과 수업 진도를 맞추기 위해 빠르게 수업을 해나간다. 그러나 일부 수강생은 실제로 메모장을 켜거나, 다운로드, 설치 등의 간단한 기능의 숙지조차 되지 않은 상태였기에, “메모장을 켜서 필기 할게요.” 라는 간단한 내용조차 따라가지 못하고, 어느 순간부터 멍하니 수업을 듣고 계셨다. 이러한 상황에서 강의 초반엔 몇 번 질문을 해서 강사님이 천천히 설명해주시긴 했지만, 계속 강의가 늦어지자 수강생분도 눈치가 보여 질문하지 못하기도, 강사님도 신경을 못쓰는 경우가 있었다. 작은 강의에서 조차 이런 일이 벌어지는 것을 본 나는 실제 대형 강의의 경우. 훨씬 더 많은 수강생이 따라가지 못할 뿐더러 쉽게 질문하지 못하는 경우가 많을 것임을 뼈저리게 느꼈다.

## 2. 문제 해결을 위한 기능

- 문제를 해결하기 위한 솔루션의 핵심 기능은 무엇인가요?
1. 학원, 학교와 제휴를 맺어 해당 커리큘럼을 미리 학습한다.
2. 만약 수강생이 강의 내용을 놓치거나, 어떻게 해야할 지 모르는 상황이 쳤다.
3. 백그라운드에서 앱이 강의를 들으며 대기하다, 화면에 떠있는 특정 버튼을 누를 경우 몇 초~분간의 내용을 요약, 해야할 것을 화면에 보여준다. → AI가 직접 맥락을 파악해 기능 실행
4. 나타난 내용을 수행 완료 시, 확인 버튼으로 해당 기능을 실행 하는 동안 놓친 내용을 요약해 설명해준다.
5. 추후 오늘 ? 버튼 누른 구간 모아 요약하기 등

## 3. 개선점

- 이 솔루션이 도입되었을 때 기대되는 개선점이 무엇인가요?
1. 장년층 혹은 실버층의 수강생이 강의를 듣다가 아주 간단한 기능의 실행에 어려움을 느낄 때, 수업 흐름 방해 없이 쉽게 해결
2. 수업에 따라가지 못하는 학생들에게 그때그때 쉽게 가이드라인 제공
3. 잠깐 놓친 부분 바로 요약 가능
4. 어려웠던 부분을 쉽게 요약해서 쉬는시간 복습, 확인 가능해 학습 효율 상승
5. 강의질(수업 진도 일정화, 밀도있는 강의 진행)상승
6. 강의 커리큘럼을 미리 업데이트를 해두는 것으로 AI의 도메인 지식 상승을 통한 내용 보충 등.
